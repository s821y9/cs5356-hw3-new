Assignment Notes

Special HTML Element

I used the <details> and <summary> elements in the cat API section. These elements create an interactive disclosure widget that lets users show or hide content without JavaScript. When clicked, the content expands or collapses, providing a clean way to organize additional information.

Special CSS Property

I implemented CSS animations using @keyframes and the animation property to create fade-in effects. I learned how keyframes define the start and end states of an animation sequence, and how to apply timing functions to control the animation speed curve.

API Integration

I used The Cat API to fetch and display random cat images. This demonstrates how to make asynchronous requests, handle JSON responses, and dynamically update the DOM with the retrieved data.
